How to build the project
---

You'll need to install [Bun](https://bun.sh/) in your machine to get this project running.

Then, you can build the JS artifact with:
```
bun run build
```

There is also a helper command to apply your extension to Spicetify in an easy way:
```
bun run apply
```

This project also comes with an integrated WebSocket server so you can test the WebSocket connection. To start it just run:
```
bun run sever
```
